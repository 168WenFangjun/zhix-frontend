### 📁 项目结构

```
frontend/
├── public/
│   └── index.html
├── src/
│   ├── components/       # 公共组件
│   ├── pages/            # 页面
│   ├── context/          # 全局状态
│   ├── utils/            # 工具函数
│   ├── config.js
│   ├── App.js
│   └── index.js
└── package.json
```

<br>

### ⚡ 三步跑起来

```bash
npm install
cp .env.local.example .env.local
npm start
```

`.env.local` 配置：

```env
REACT_APP_API_URL=http://localhost:8080
REACT_APP_PAYMENT_API_URL=http://localhost:8081
```

> 启动后访问 → `http://localhost:3000`
